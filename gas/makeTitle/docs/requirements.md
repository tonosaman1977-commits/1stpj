# 研究タイトル生成エンジン 要件定義書

## 1. プロジェクト概要

### 成果目標
研究概要テキストを入力すると、学術・一般・SNS向け計15案のタイトルをGemini APIで自動生成し、カテゴリ別・全一括でコピーできるGAS WebApp。

### 成功指標（定量）
1. 入力から生成完了まで30秒以内（Gemini API応答依存）
2. 3カテゴリ × 5案 = 計15タイトルを1リクエストで生成
3. 入力上限20,000文字に対応

### 成功指標（定性）
1. 研究者がタイトル案ゼロから悩む時間をゼロにできる
2. 誇張・断定なしのルール適用で安心して使える
3. ワンクリックコピーで他ツールへ即転用できる

---

## 2. システム全体像

### 機能一覧
| # | 機能 | 説明 |
|---|------|------|
| 1 | 研究概要入力 | 10〜20,000文字のテキスト入力 |
| 2 | タイトル生成 | Gemini APIで3カテゴリ×5案を一括生成 |
| 3 | カテゴリ別コピー | 各カテゴリの5案をまとめてコピー |
| 4 | 全一括コピー | 15案すべてをまとめてコピー |
| 5 | エラー表示・リトライ | API失敗・バリデーションエラー時にメッセージとリトライボタンを表示 |

### 認証要件
なし（GAS WebApp として公開）

### スコープ外
- 英語タイトル生成
- SEO最適化・キーワード分析
- 自動投稿
- 学術分野別最適化
- 任意入力（audience / keywords）は将来拡張

## 3. ページ詳細仕様

### P-001: タイトル生成ページ

**目的**: 研究概要を入力してタイトル案を生成・コピーする

**URL**: GAS WebApp エンドポイント（/）

**画面構成**
```

研究概要を入力してください
[テキストエリア: 最低10文字、最大20,000文字]

[タイトル生成ボタン]  ← クリックでローディングスピナー表示

─────────────────────────
研究タイトル（学術向け）
① ...
② ...
③ ...

⑤ ...

─────────────────────────
一般向けタイトル
① ...
② ...
③ ...
④ ...
⑤ ...
[カテゴリコピー]

─────────────────────────
SNSタイトル
① ...
② ...
③ ...

⑤ ...
[カテゴリコピー]

─────────────────────────
[全カテゴリまとめてコピー]

─────────────────────────

エラーメッセージ
[リトライ]
```

**処理フロー**
1. 研究概要入力 → クライアントバリデーション（10文字以上）
2. タイトル生成ボタンクリック → ローディングスピナー表示
3. `google.script.run.generateTitles(text)` でGASサーバー呼び出し
4. GASサーバー → Gemini API（1回呼び出し、3カテゴリ一括生成）
5. JSON応答をクライアントで受け取り → カテゴリ別に表示
6. エラー時 → エラーメッセージ + リトライボタン表示

## 5. 出力仕様

### JSONスキーマ（Gemini responseSchema）
```json
{
  "type": "object",
  "properties": {
    "academic": {
      "type": "array",
      "items": { "type": "string" },
      "description": "学術向けタイトル5案"
    },
    "general": {
      "type": "array",
      "items": { "type": "string" },
      "description": "一般向けタイトル5案"
    },
    "sns": {
      "type": "array",
      "items": { "type": "string" },
      "description": "SNS向けタイトル5案"
    }
  },
  "required": ["academic", "general", "sns"]
}
```

### 生成ルール
| カテゴリ | トーン | 文字数目安 | 特徴 |
|---------|--------|-----------|------|
| 学術向け | 客観・簡潔 | 20〜50文字 | 学術的・論文タイトル形式 |
| 一般向け | 分かりやすい・やや説明的 | 20〜40文字 | 専門用語を減らす・興味喚起 |
| SNS向け | 短い・フック重視 | 15〜30文字 | 興味を引く・拡散しやすい |

**共通禁止事項**
- 元研究内容にない事実の追加
- 誇張表現
- 効果の断定

## 6. データ設計

サーバーサイド永続化なし（ステートレス）。
入力テキストはGASサーバー → Gemini APIへの都度送信のみ。

## 7. セキュリティ要件

| 項目 | 対応 |
|------|------|
| APIキー管理 | PropertiesService.getScriptProperties() に保存、コードへのハードコード禁止 |
| 入力サニタイズ | 文字数チェック（クライアント + サーバー） |
| HTTPS | GAS WebApp はデフォルトでHTTPS |
| エラー隠蔽禁止 | Gemini APIエラーはUI表示。フォールバックで隠蔽しない |
