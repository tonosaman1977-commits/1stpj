# 多言語研究説明エンジン 要件定義書

## 1. プロジェクト概要

### 成果目標
研究概要テキストを入力するだけで、やさしい日本語・英語・留学生向け説明＋選択式5言語の説明を即時生成し、
研究者の多言語発信コストを大幅削減する独立GAS WebApp。

### 成功指標（定量）
| # | 指標 | 目標値 |
|---|------|--------|
| 1 | 説明文生成時間 | 30秒以内 |
| 2 | 入力文字数対応範囲 | 20〜20,000文字 |
| 3 | 固定生成セクション数 | 3種（やさしい日本語・英語・留学生向け） |
| 4 | 選択式追加言語数 | 5言語（中国語・韓国語・ベトナム語・スペイン語・フランス語） |
| 5 | やさしい日本語の長さ | 200〜350文字 |
| 6 | 英語説明の長さ | 120〜200 words |
| 7 | 留学生向け説明の長さ | 200〜350文字 |

### 成功指標（定性）
| # | 指標 |
|---|------|
| 1 | 元研究内容に忠実（事実追加・誇張・効果断定なし） |
| 2 | 各対象（留学生・海外研究者・一般読者）に適した文体 |
| 3 | コピーボタンで即座に使える実用性 |
| 4 | エラー時に原因が分かるUI |

## 4. ページ詳細仕様

### P-001: 多言語説明生成ページ（単一ページ完結）

**目的**: 研究概要を入力し多言語説明を生成・コピーする

**構成要素**

| 要素 | 仕様 |
|------|------|
| タイトル | 多言語研究説明エンジン |
| 入力エリア | textarea、placeholder付き、20〜20,000文字 |
| 追加言語チェックボックス | 中国語・韓国語・ベトナム語・スペイン語・フランス語 |
| 生成ボタン | ローディング中は無効化＋スピナー表示 |
| 固定出力セクション | やさしい日本語 / English Explanation / 留学生向け説明 |
| 追加言語出力セクション | 選択した言語のみ表示 |
| コピーボタン | 各セクションに1つ |
| エラー表示 | エラーメッセージ＋リトライボタン |

**処理フロー**
```
1. ユーザーが研究概要を入力
2. 追加言語をチェックボックスで選択（任意）
3. 「生成する」ボタンクリック
4. クライアント側バリデーション（文字数チェック）
5. google.script.run.generateExplanations(text, selectedLanguages) 呼び出し
6. ローディングスピナー表示・ボタン無効化
7. GASサーバー: Gemini API呼び出し（固定3種＋選択言語を一括生成）
8. 結果をJSONで返却
9. 各セクションに結果を表示
```

| パラメータ | 型 | 説明 | 制限 |
|-----------|-----|------|------|

### 任意入力
| パラメータ | 型 | 説明 |
|-----------|-----|------|

### selectedLanguages の値

| `korean` | 韓国語 |
| `vietnamese` | ベトナム語 |
| `spanish` | スペイン語 |
| `french` | フランス語 |

| ベトナム語 | ベトナム語話者留学生 | 200〜350文字相当 | 自然なベトナム語 |

## 7. 生成ルール（プロンプト設計）

### 共通禁止事項
- 元の研究内容にない事実の追加
- 効果の断定
- 誇張表現

### カテゴリ別ルール
| カテゴリ | ルール |
|---------|--------|
| やさしい日本語 | 1文を短くする・難しい言葉は言い換える・中学生でも読めるレベル |
| English | シンプルな文構造・過度な専門表現を避ける・読みやすさ優先 |
| 留学生向け | 研究分野を明確にする・日本の研究環境に触れてもよい・専門分野の興味を引く説明 |
| 各外国語 | 自然な表現・専門用語は必要に応じて説明 |

## 8. セキュリティ要件

- 入力値サニタイゼーション（クライアント・サーバー両方で文字数チェック）
- APIキーはPropertiesServiceで管理（コードへのハードコード禁止）
- GAS WebApp: HTTPS強制（GASが自動対応）
